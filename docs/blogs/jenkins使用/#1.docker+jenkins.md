# docker+jenkins
> 自动化构建工具链 

## docker
> 容器 非常轻量

### install
官方脚本安装
```sh
  curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
```
docker常用命令
```sh
docker ps
docker images
docker pull
docker run
```
docker名称参数
```sh
-t
-d
```
